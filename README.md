### 运维平台功能简介

**功能点：**
```
1.用户的增删改查
2.用户权限的增删改查
3.阿里云主机的信息获取与展示
4.登录系统与权限控制
```
一、用户管理系统

>1.用户基本信息展示，权限展示
>
>2.基于权限来限制用户的页面内容RBAC
>
>3.资产管理，标签编辑，主机信息
>
>4.基于阿里云API来操作主机
